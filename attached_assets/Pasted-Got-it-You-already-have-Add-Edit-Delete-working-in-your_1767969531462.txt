# âœ… STEP 3: USE ONE ROUTE FOR ALL DROPDOWN TOPICS

### ğŸ”¹ This route makes **ALL dropdown pages work**

```js
app.get("/market/:category", (req, res) => {
  const category = req.params.category;
  const posts = JSON.parse(fs.readFileSync("database.json"));

  const filteredPosts = posts.filter(
    post =>
      post.category === category &&
      post.status === "published"
  );

  let html = `
    <h1 style="color:#137333">
      ${category.replace(/-/g, " ").toUpperCase()}
    </h1>
  `;

  if (filteredPosts.length === 0) {
    html += "<p>No articles available.</p>";
  }

  filteredPosts.forEach(post => {
    html += `
      <article style="margin-bottom:20px;">
        <h2>${post.title}</h2>
        <p>${post.content.substring(0, 150)}...</p>
        <a href="/post/${post.id}">Read more</a>
      </article>
      <hr>
    `;
  });

  res.send(html);
});
```
