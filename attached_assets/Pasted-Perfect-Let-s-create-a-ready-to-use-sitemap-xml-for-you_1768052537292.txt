# ‚úÖ How to Use in Replit

```

```

2. Paste the content above
3. Save the file
4. Your sitemap will now be live at:

```
https://your-project-name.username.repl.co/sitemap.xml
```

5. Update your `robots.txt` to include it (we already added the line):

```txt
Sitemap: https://yourwebsite.com/sitemap.xml
```

6. Submit it in **Google Search Console** ‚Üí ‚ÄúSitemaps‚Äù section
